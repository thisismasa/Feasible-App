🎯 WHAT IS CURSOR?
═══════════════════════════════════════════════════════════════════════════════

Cursor is an AI-powered code editor (like VS Code) that integrates:
✅ Claude (Anthropic) ⭐
✅ GPT-4
✅ Other AI models

Perfect for coding with AI assistance!

METHOD 1: Use Cursor Pro (EASIEST - NO API KEY NEEDED!)
────────────────────────────────────────────────────────────────

Step 1: Download Cursor

🔑 METHOD 2: Use Your Own API Key (More Control)
═══════════════════════════════════════════════════════════════════════════════

Step 1: Get Anthropic API Key
──────────────────────────────
1. Go to: https://console.anthropic.com/
2. Sign up / Log in
3. Create API key (starts with sk-ant-...)
4. Copy your key


Step 2: Add Key to Cursor
──────────────────────────────
Press: Ctrl + , (Settings)
Search: "anthropic"
Paste your API key
Save


Step 3: Select Claude
──────────────────────────────
Press: Ctrl + K
Select: Claude 3.5 Sonnet
Done! ✅

🎮 HOW TO USE CLAUDE IN CURSOR
═══════════════════════════════════════════════════════════════════════════════

KEYBOARD SHORTCUTS
──────────────────────────────

Ctrl + L     → Open Claude chat
Ctrl + K     → Inline code edit
Ctrl + I     → Composer (multi-file)
Ctrl + ,     → Settings


EXAMPLE USAGE
──────────────────────────────

1. Chat Mode (Ctrl + L)
───────────────────────
Press: Ctrl + L
Type: "Create a Flutter login widget"
Claude: Creates the widget code for you


2. Inline Edit (Ctrl + K)
───────────────────────
Select code
Press: Ctrl + K
Type: "Add error handling"
Claude: Updates your code


3. Composer (Ctrl + I)
───────────────────────
Press: Ctrl + I
Type: "Create login, home, and profile screens"
Claude: Creates multiple files at once


4. Context-Aware
───────────────────────
Type: @workspace Explain this Flutter app
Type: @file What does this do?
Type: @code Review this code

🔧 CURSOR SETTINGS FOR CLAUDE
═══════════════════════════════════════════════════════════════════════════════

Press: Ctrl + Shift + P
Type: "Preferences: Open User Settings (JSON)"

Add this:
─────────

{
  // Use Claude as default
  "cursor.chat.defaultModel": "claude-3-5-sonnet-20241022",

  // API Key (if using your own)
  "anthropic.apiKey": "sk-ant-YOUR-KEY-HERE",

  // Enable features
  "cursor.chat.enabled": true,
  "cursor.autocomplete.enabled": true
}
