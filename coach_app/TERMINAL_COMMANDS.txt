💻 RUN ON DIFFERENT PLATFORMS
─────────────────────────────────────────────────────────────────────────────────

Run on Chrome (fastest):
flutter run -d chrome

Run on Windows Desktop:
flutter config --enable-windows-desktop
flutter run -d windows

Run on Android (needs emulator):
flutter run
